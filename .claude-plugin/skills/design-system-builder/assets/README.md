# Templates

This directory contains all templates for the Design System Builder skill.

## Available Templates

### Design Tokens
- `design-tokens-template.json` - Base token structure
- `css-custom-properties.css` - CSS variables format
- `scss-variables.scss` - SCSS variables format
- `tailwind-config.js` - Tailwind CSS config
- `style-dictionary-config.js` - Style Dictionary setup

### Components
- `component-template.tsx` - React/TypeScript component
- `component-story-template.js` - Storybook story
- `component-spec-template.md` - Component specification
- `button-spec-template.md` - Button-specific spec

### Configuration
- `storybook-main.js` - Storybook main config
- `storybook-preview.js` - Storybook preview config
